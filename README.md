# Структуры данных

| Структура данных | Доступ              | Поиск               | Вставка             | Удаление            |
| ---------------- | ------------------- | ------------------- | ------------------- | ------------------- |
| Связный список   | 🟡O(n)              | 🟡O(n)              | 🟢O(1)              | 🟢O(1)              |
| Бинарное дерево  | 🟢O(log n) - 🟡O(n) | 🟢O(log n) - 🟡O(n) | 🟢O(log n) - 🟡O(n) | 🟢O(log n) - 🟡O(n) |

# О-Большое

| Цвет | Название                | Обозначение | Комментарий             |
| ---- | ----------------------- | ----------- | ----------------------- |
| 🟢   | Константная             | O(1)        | Самая быстрая           |
| 🟢   | Логарифмическая         | O(log n)    | Очень медленный рост    |
| 🟡   | Линейная                | O(n)        | Растёт пропорционально  |
| 🟠   | Линейно-логарифмическая | O(n log n)  | Быстрая для больших n   |
| 🔴   | Квадратичная            | O(n²)       | Замедляется заметно     |
| 🔴   | Экспоненциальная        | O(2ⁿ)       | Растёт крайне быстро    |
| 🔴   | Факториальная           | O(n!)       | Почти всегда непригодна |
